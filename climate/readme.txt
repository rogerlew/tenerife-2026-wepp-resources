Source: Cabildo de Tenerife (https://www.agrocabildo.org/agrometeorologia_estaciones.asp)


Station types:
	Type A: outdoor temperature, precipitation, relative humidity, radiation, wind speed and direction and evaporation. Indoor temperature, relative humidity, radiation and evaporation (greenhouse).
	Type B: outdoor temperature, precipitation, relative humidity, radiation, wind speed and direction, evaporation and leaf wetness.
	Type C: outdoor temperature, precipitation, relative humidity, radiation, wind speed and leaf wetness.
	Type D: outdoor temperature, precipitation, relative humidity, radiation, wind speed and direction.
	Type E: outdoor temperature, precipitation, relative humidity, radiation and wind speed.
